## Recording Checklist

Before recording:
- [ ] Chrome at 100% zoom, no bookmarks bar, dark OS theme
- [ ] Clear browser history / use fresh profile so no cached data
- [ ] Resize browser to 1920x1080 exactly
- [ ] OBS: 1920x1080, 30fps, MP4, 8Mbps target bitrate
- [ ] Mic test: record 10 seconds, check levels
- [ ] Run one Anthropic mission to pre-warm the Bright Data connections

During recording:
- [ ] Move cursor deliberately — no fast flicks
- [ ] Pause 0.5s before clicking any important UI element (editors cut here)
- [ ] Let the agent stream run without interruption — the animation IS the demo
- [ ] When panning the BD panel, hold 1–2 seconds on each card

Post-production:
- [ ] Add title card at 0:00 (black + text animation) using DaVinci Resolve or CapCut
- [ ] Add lower thirds for each section: "TRACK 1 / GTM INTELLIGENCE", "TRACK 2 / SUPPLY CHAIN", "TRACK 3 / THREAT SURFACE", "POWERED BY BRIGHT DATA"
- [ ] Add subtle music: ambient/electronic, low volume, under voiceover
- [ ] Export: H.264, 1080p, <200MB for upload
- [ ] Upload to YouTube (unlisted) → paste link in submission form
